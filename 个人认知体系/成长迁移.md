# 因果与系统思维

从“谁错了”的追责/共情，到“流程/系统哪里出了问题”。打破对错的认知、而是建立因果的认知。

**实践**：考虑当前阶段需要着重解决的问题，并按照合适的颗粒度划分因果关系。根据果，付诸因的实践。


# 边界管理

1. 区分自己的课题和他人的课题。
2. 对值得的人进行共情。
3. 勇敢表达自己的想法，但也注意尊重他人的边界。


# 情绪与行动解耦

在任何强烈情绪下，一定要先处理情绪，再行动，否则会造成不可挽回的后果。


# 深度自我觉察

了解自己的优势和劣势，能够更好地搭配团队、与人合作。


# 不评判，而是理解

理解和洞察他人行为背后的动机、局限和需求，而非进行抱怨、评判。


# 自我的完整性与阶段局限性

我承认自己心中有嫉妒、自私、恐惧的种子，但**我决定不任由这些种子生长为支配我的藤蔓**。我选择给另一颗种子——真诚、责任、善意——浇水施肥。

我被迫凝视了深渊，也审视了自己在深渊边的倒影。我理解了人在极端痛苦下可能有多么不堪，也理解了从不堪中生长出觉知的可能性。这种认知，比单纯的黑白判断要厚重得多。
