# 2. 自我要求
## 日常相处

1. **学会赞美：** 从身边人开始，有意识地、具体地表达你的看见和欣赏。“你做的这件事，体现了你的XX特质，我觉得很棒。” 这会重塑你的表达习惯。
2. **情绪边界** ：未来痛苦、失落等负面情绪的想要和伴侣宣泄时，这是我的课题，我需要先自己处理，然后寻求支持，而非要求对方负责。
3. **沟通方式**：理解对方表达沟通方式背后的真正需求，而非只关注表面问题。


# 核心模式

## 爱的语言

双方表达爱的语言可能是不同的：

- **不成熟的爱**是对方应该知道我想要什么，并用我想要的方式给我。
- **成熟的爱**是学习对方爱的“语言“去表达，同时也会教会对方理解我的“语言”
