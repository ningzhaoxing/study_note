# 单测思路

## hack仓储层数据

## 执行待测对象
`When`: 当执行xxx的时候

## 预期结果
`Then`: 判断是否符合预期结果

# 自定义hack mock

假设 `directory_service.IDirectoryService` 接口没有生成对应的 `mock` 方法。
```go
type mockDirectorySvc struct {
	directory_service.IDirectoryService
}

Mock((*mockDirectorySvc).MGetTenant).Return(&directory_service.TenantEntity{}, nil).Build()

mcpServerApp := &MCPServerApplication{
	directoryService:  &mockDirectorySvc{},
}
```
