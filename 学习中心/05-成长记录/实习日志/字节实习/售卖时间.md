个人生产总值 = 时间单价 ✖️ 能产生价值的时间

如果想提高个人生产总值，就需要提高你的 **时间单价** 或 **能产生价值的时间** 。

具体有以下几种方式：
# 1. 零售时间

通过“勤奋”来增加“能产生价值的时间”，但这种方式是有极限的，一天只有24小时。
并且这种方式兑换收入还要被打折，购买他时间的老板中间也会赚钱。

# 2. 批发时间

把你创造出的价值，同时卖给很多人。

> 比如演讲，面对1个人演讲2小时，和面对100个人演讲2小时，所产生的价值的时间翻了100倍。

这就相当于把时间批发出去了。

# 3. 买卖时间

企业为什么花钱请你来工作。

1. 你在某方面更专业
	如果让一个人从头开始学的话，时间成本太高，结果还不一定会比你做得好。因此，需要你快速帮助企业补齐这个能力。
2. 你能成为他们的帮手
	有一类事情，难度不大，但是又不得不做，还很花时间。为了让企业的核心人员能够专心做有价值的事情，节省他们的时间，就需要这类员工去做这些事情。

所以，任何公司主要就有两大类员工：

1. 创造价值的员工
2. 提升效率的员工

因此，如果你暂时是一名打工人。那你的能力要足够企业去花钱买你的时间。

如果你能创造价值，那么就需要买卖他人的时间帮你提高效率，以帮助你专心创造价值。

# 4. 收时间税

用户在你的平台上活跃，用户的数量会影响你的税收。

平台成功的关键是，如何为用户赋能？

> 个人生产总值 = **时间单价** ✖️ **能产生价值的时间**

选择你的平台，就是希望通过平台帮助他的个人总值变得更高。
因此，你的平台要想办法提高他的 **时间单价** 或 **提高他能产生价值的时间** 。
