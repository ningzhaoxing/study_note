# Golang 后端开发面试题汇总

> 来源：基于牛客网面经总结

## 一、Go 语言基础

### 1.1 数据结构与算法

1. **切片 vs 数组**
   - 数组是值类型，切片是引用类型
   - 切片底层结构：指针、长度、容量
   - 切片扩容机制：容量 < 1024 时翻倍，>1024 时增长 1.25 倍

2. **Map 实现**
   - 底层使用 hash 表
   - 冲突解决：链地址法
   - 线程不安全，并发读写需加锁或使用 sync.Map

3. **Context 用途**
   - 超时控制
   - 取消信号传递
   - 请求作用域数据存储

### 1.2 并发编程

1. **Goroutine 调度**
   - GMP 模型：Goroutine、Machine、Processor
   - 抢占式调度
   - 工作窃取算法

2. **Channel 类型**
   - 有缓冲 vs 无缓冲
   - select 多路复用
   - close 操作和 range 遍历

3. **Sync 包**
   - sync.WaitGroup：等待一组 goroutine 完成
   - sync.Mutex：互斥锁
   - sync.RWMutex：读写锁
   - sync.Once：只执行一次
   - sync.Pool：对象池

4. **常见并发问题**
   - 竞态条件：使用 go race 检测
   - 死锁：避免循环等待
   - goroutine 泄漏：正确使用 context 取消

### 1.3 内存管理

1. **垃圾回收**
   - 三色标记法
   - 写屏障
   - 分代回收（Go 1.21+）

2. **内存分配**
   - TCMalloc 思想
   - mspan、mcache、mcentral、mheap
   - 逃逸分析

### 1.4 接口与反射

1. **接口实现**
   - duck typing
   - 接口值的底层结构：type + value
   - 空接口 vs 非空接口

2. **反射使用**
   - reflect.Type 和 reflect.Value
   - 反射三定律
   - 性能开销
