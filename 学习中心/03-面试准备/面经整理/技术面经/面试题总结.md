## 📚 内容分类

### 一、Go 语言基础

#### 1. 数据结构与实现（15题）

**高频考点：**
1. ✅ Slice 扩容机制
2. ✅ Map 底层实现
3. ✅ Channel 数据结构
4. ✅ Interface 内部结构
5. ✅ Defer 执行顺序

**详解：**

**Slice 扩容**
```go
// 容量 < 1024：翻倍
// 容量 >= 1024：增长 1.25 倍
// Go 1.18+ 优化：对齐内存
```

**Map 实现**
- 底层：hash 表
- 冲突：拉链法
- 扩容：渐进式（翻倍）
- 并发：不安全，需加锁

**Channel**
- 结构：环形缓冲区 + 锁
- 类型：有缓冲 / 无缓冲
- 操作：send、recv、close

#### 2. 并发编程（20题）

**核心概念：**
1. ✅ GMP 调度模型
2. ✅ Channel 使用场景
3. ✅ Sync 包组件
4. ✅ Context 传播

**GMP 模型：**
- G：Goroutine（用户态线程）
- M：Machine（系统线程）
- P：Processor（逻辑处理器）
- 调度：抢占式 + 工作窃取

**Sync 包：**
- `sync.Mutex`：互斥锁
- `sync.RWMutex`：读写锁
- `sync.WaitGroup`：等待组
- `sync.Once`：单次执行
- `sync.Pool`：对象池

#### 3. 内存管理（10题）

**垃圾回收：**
1. ✅ 三色标记法
2. ✅ 写屏障
3. ✅ 分代回收（Go 1.21+）

**内存分配：**
- TCMalloc 思想
- mspan、mcache、mcentral、mheap
- 逃逸分析

## 💡 备考建议

### 1. 理论学习（40%）

**Go 语言：**
- 深入理解底层原理
- 掌握并发编程模式
- 熟悉标准库实现

**数据库：**
- MySQL 索引原理
- 事务和锁机制
- SQL 优化技巧

**Redis：**
- 数据结构底层实现
- 持久化方案
- 高可用架构

### 2. 刷题练习（30%）

**LeetCode：**
- Hot 100 必刷
- 每日保持 2-3 题
- 限时训练（30分钟/题）
- 总结解题模板

**分类练习：**
- 链表：掌握常见操作
- 二叉树：熟练递归
- 动态规划：理解状态转移
- 回溯：掌握模板

### 3. 系统设计（20%）

**准备方案：**
- 秒杀系统
- 短链接服务
- IM 即时通讯
- 排行榜系统

**设计思路：**
- 明确需求
- 分阶段实现
- 考虑扩展性
- 画图表达

### 4. 项目经验（10%）

**STAR 法则：**
- S：背景（Situation）
- T：任务（Task）
- A：行动（Action）
- R：结果（Result）

**准备内容：**
- 技术难点
