- **岗位职责**：
  
  作为核心开发成员，负责 AIcanTARA 平台 AI Agent 模块的架构设计与工程落地，主导资产识别、攻击树生成等核心 AI 分析能力以及 TARA 报告导出的后端实现。

1. **Agent 工具体系与复用架构**：基于 Eino ADK 构建多模块 Agent 系统（资产识别、攻击树生成、风险管理等），采用 **Skill Middleware 架构**实现工具按领域模块化组织与声明式注册，支持核心能力（项目上下文、资产查询）**跨 Agent 复用**，降低新 Agent 接入成本

4. **试用版功能管控**：基于 JWT Token 扩展 plan 字段实现版本路由，设计 LicenseConfig 抽象层封装版本差异，Service 层仅依赖 License 对象而非版本标识，预留多维度功能限制的扩展能力
