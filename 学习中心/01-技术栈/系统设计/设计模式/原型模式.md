# 概念

原型模式：通过拷贝复制对象，而不是通过重新构造复制对象。

  ```plantuml
  class Client {}
  
  interface Prototype{
	  Clone()
  }
  
  class ConcretePrototype1 {}
  
  class ConcretePrototype2 {}
  
  ConcretePrototype1 --u|> Prototype
  
  ConcretePrototype2 --u|> Prototype
  
  Client --> Prototype
  ```

在 `clone()` 中实现对当前对象的深拷贝。


# 应用场景

需要对实例进行复制而不是重新构造的场景。
