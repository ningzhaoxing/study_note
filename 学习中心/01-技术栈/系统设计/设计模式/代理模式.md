# 概念

代理模式（Proxy），为其他对象提供一种代理以控制对这个对象的访问。

```plantuml
interface subject {
	Request()
}

class realSubject {
}

class proxy {
	beforeRequest()
	afterRequest()
}

realSubject --u|> subject
proxy --u|> subject

proxy ..> realSubject

class client {
}

client --> subject
```

# 应用场景

访问控制，对外不暴露真实主题，真实主题由代理来创建。
