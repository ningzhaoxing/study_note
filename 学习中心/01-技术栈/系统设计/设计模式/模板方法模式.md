# 概念
模板方法模式， 父类通过**模板方法** **定义算法执行框架**、**步骤**，子类实现父类需要**被动态定义的方法**。

```plantuml

}

class ConcreteClass {
PrimitiveOperation1()

}

ConcreteClass --u|> Template
```

go 语言实现

`Game.go`

```go
type IGame interface {  
    Init()  
    Start()  
    End()  
    Play()  
}  
  
type Game struct {  
    IGame  
}  
  
func (b *Game) Play() {  
    b.Init()  
    b.Start()  
    b.End()  
}  
  
type Football struct {  
    Game  
}  
  
func NewFootball() Game {  
    f := &Football{}  
    f.IGame = f  
    return f.Game  
}  
  
func (f *Football) Init() {  
    fmt.Println("Football Game Initialized!")  
}  
  
func (f *Football) Start() {  
    fmt.Println("Football Game Started!")  
}  
  
func (f *Football) End() {  
    fmt.Println("Football Gaml Ended!")  
}
```

`client.go`

```go
func main() {
	footballGame := NewFootball()
	footballGame.Play()
}
```
