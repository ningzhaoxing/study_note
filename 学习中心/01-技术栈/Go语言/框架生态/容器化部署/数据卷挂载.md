# 数据卷
数据卷是一个虚拟目录，是***容器内目录***与***宿主机目录***之间映射的桥梁。
![[20250222131638.png]]据卷常见命令
```linux
# 查看数据卷
docker volume ls
# 删除数据卷
docker volume rm
# 查看数据卷详情
docker volume inspect 数据卷名称
# 删除未使用数据卷
docker volume prune 数据卷名称
```

# 修改nginx中的静态资源html文件
## 将数据卷挂载到容器目录
在创建容器时，利用 `-v 数据卷名:容器内目录`完成挂载。
若容器创建时，发现挂载的数据卷不存在，会自动创建。
```linux
docker run -d --name nginx -p 80:80 -v html:/usr/share/nginx/html nginx
```
## 需要宿主机对应目录html文件
修改目录
```linux
/var/lib/docker/volumes/数据卷名称/_data
```
