由于每次容器重启，其IP地址会发生变化。所以，无法直接通过IP地址进行互连。

docker提供了自定义网络，使之可以通过**容器名**进行互连。

# 常用命令

## 创建网络
`docker network create 网络名`

## 查看网络列表
`docker network ls`

## 将容器加入网络
`docker network connect 网络名 容器名`

## 创建容器时，直接加入网络
`docker run -d --name 容器名 -p 8080:8080 --network 网络名 镜像名`
