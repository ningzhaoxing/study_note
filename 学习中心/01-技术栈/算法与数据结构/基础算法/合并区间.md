## 解题思路
将区间按照左端点进行排序，再依次遍历每个区间，若当前区间存在于上一区间，则将该区间加入。若不在该区间或当前ans为空，则添加到ans中。

# AC 代码
```go
func getMax(a, b int) int {  
    if a > b {  
       return a  
    }  
    return b  
}  
  
func merge(intervals [][]int) [][]int {  
    slices.SortFunc(intervals, func(a, b []int) int {  
       return a[0] - b[0]  
    })    ans := make([][]int, 0)  
  
    for _, a := range intervals {  
       m := len(ans)  
       // 在区间内，进行合并  
       if m > 0 && ans[m-1][1] >= a[0] {  
          ans[m-1][1] = getMax(ans[m-1][1], a[1])  
       } else {  
          // 不在区间内，则将新区间添加  
          ans = append(ans, a)  
       }    }    return ans  
}
```


# 代码技巧
## 自定义排序
```go
// 从小到大排序
slices.SortFunc(arr, func(a, b int) int {
	return a-b
})
```

## 判断一个区间是否在另一个区间内
只需判断区间左端点是否存在于另一区间内。
