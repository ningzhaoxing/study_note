# 解题思路
可以递归搜索所有情况。

递归思路：
如果搜索到当前元素，是字母，则进行大写或小写转化或不进行转化(2种选择)，然后继续递归；
如果搜索到当前元素，是数字，则直接进行下次递归搜索。

递归结束标志：
当递归到叶子节点，即搜索到字符串最后一个元素，即结束当前深搜路径，并将递归结果加入答案列表。

回溯：
如果进行大小写转化，则在下层递归搜索结果出来后，将其重新进行大小写转化(就是原来元素本身了)。

# AC代码
```go
var res []string

var m map[string]struct{}

  

func letterCasePermutation(s string) []string {

    res = make([]string, 0)

    m = make(map[string]struct{})

    by := []byte(s)

    dfs(by, 0)

    return res

}

  

func isBig(b uint8) bool {

    if b > 'A' && b < 'B' {

        return true

    }

    return false

}

  

func dfs(s []byte, cur int) {

    if _, ok := m[string(s)]; !ok {

        res = append(res, string(s))

        m[string(s)] = struct{}{}

    }

  

    if len(s) == cur {

        return

    }

  

    if !unicode.IsLetter(rune(s[cur])) {

        dfs(s, cur+1)

    } else {

        // 进行大小写转化

        s[cur] ^= ' '

        dfs(s, cur+1)

        s[cur] ^= ' '

        // 不进行大小写转化

        dfs(s, cur+1)

    }

}
```

# 代码技巧
## 字母大小写转化
可以通过对`' '`进行异或运算,即`A == (a ^= ' ')`
## go判断是否为字符类型
`unicode.IsLetter()`
