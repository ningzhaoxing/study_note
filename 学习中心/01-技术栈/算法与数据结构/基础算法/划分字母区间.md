# 解题思路
只需要找出每个字母所在的是最后一个位置的索引，然后在遍历s的过程中进行更新右端点和区间合并，直到遍历到区间右端点，则是一个答案区间，直到遍历字符串s结束。

# AC代码
```go
func partitionLabels(s string) []int {  
    last := [26]int{}  
    for i, v := range s {  
       last[v-'a'] = i  
    }  
  
    start, end := 0, 0  
    ans := make([]int, 0)  
    for i, i2 := range s {  
       end = max(end, last[i2-'a'])  
       if i == end {  
          ans = append(ans, end-start+1)  
          start = end + 1  
       }  
    }    return ans  
  
}
```

# 代码技巧
