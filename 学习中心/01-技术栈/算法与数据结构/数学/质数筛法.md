# 埃氏筛法
对于求`1~n`中的所有质数，从小到大枚举每个数，
- 如果当前数没划掉，必定是质数，记录该质数；
- 枚举当前质数的倍数，必定是合数，划掉合数。
```go
// Eratosthenes 埃氏筛法  
func Eratosthenes(n int) []int {  
    isPrim := make([]bool, n+1)  
    prims := make([]int, 0)  
  
    isPrim[0], isPrim[1] = false, false  
    for i := 2; i <= n; i++ {  
       isPrim[i] = true  
    }  
	// 筛至平方根即可
    for i := 2; i*i <= n; i++ {  
       if isPrim[i] {  
          for j := i * i; j <= n; j += i {  
             isPrim[j] = false  
          }  
       }    
    }  
    for i := 2; i <= n; i++ {  
       if isPrim[i] {  
          prims = append(prims, i)  
       }
    }  
    return prims  
}
```
