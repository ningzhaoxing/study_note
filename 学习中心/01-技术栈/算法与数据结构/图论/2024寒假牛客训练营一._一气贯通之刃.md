# 题解思路
> 简单路径：
> 其经过的顶点和边都不相同。

题目要求找到一条可以经过树所有节点的***简单路径***。
如果一个节点存在`n(n>=3)`个子节点，那么要经过树的所有节点就要经过这个节点`(n-1)`次。

因此，该树一定是一个链，那么只需要找到该链的左右端点即可。
因为链的端点节点的度一定为`1`，所以只需要计算每个节点的度即可。

# AC代码
```go
#include <bits/stdc++.h>
#define int long long
using namespace std;
void solve() {
    int n;
    cin >> n;
    vector<int> in(n+1);
    for (int i = 1; i < n; ++i) {
        int u,v;
        cin >> u >> v;
        in[u]++;in[v]++;
    }

    vector<int> ans;
    for (int i = 1; i <= n; ++i) {
        if (in[i] == 1) ans.push_back(i);
        if (in[i] > 2) {
            cout << -1 << endl;
            return;
        }
    }
    int ans1 = ans.at(0);
    int ans2 = ans.at(1);
    cout << ans1 << " " << ans2 << endl;
}

signed main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int t = 1;
//    cin >> t;
    while (t--) solve();
}
```
