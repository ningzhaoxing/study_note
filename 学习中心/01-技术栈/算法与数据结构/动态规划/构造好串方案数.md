## 解题思路
可以将问题抽象为：
到达`low`到`high`阶梯方案数总和。

## 递归公式
$$
dfs(i) = dfs(i-zero) + dfs(i-one)
$$
$$
sum = \sum_{{i=low}}^{{high}}dfs(i)
$$
## AC代码
```go
func countGoodStrings(low int, high int, zero int, one int) int {  
    mod := 1_000_000_007  
    f := make([]int, high+1)  
  
    f[0] = 1  
    for i := 1; i <= high; i++ {  
       if i-zero >= 0 {  
          f[i] = f[i-zero]  
       }  
       if i-one >= 0 {  
          f[i] = (f[i] + f[i-one]) % mod  
       }  
    }  
    sum := 0  
    for i := low; i <= high; i++ {  
       sum = (sum + f[i]) % mod  
    }  
    return sum  
}
```
