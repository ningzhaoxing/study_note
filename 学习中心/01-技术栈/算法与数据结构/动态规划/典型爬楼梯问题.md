# 解题思路
## 思考子问题
如果爬到第`i`阶楼梯，那么它可能是从`i-1`层楼梯或`i-2`层楼梯爬上来的。
## 递归公式
$$
dfs(i) = dfs(i-1) + dfs(i-2) \ if\ i>=2
$$
$$
dfs(0) = 1 \

dfs(1) = 1
$$
## AC代码
```go
func climbStairs(n int) int {  
    if n == 1 {  
       return 1  
    }  
    if n == 2 {  
       return 2  
    }  
  
    dp := make([]int, n+1)  
    dp[1] = 1  
    dp[2] = 2  
    for i := 3; i <= n; i++ {  
       dp[i] = dp[i-1] + dp[i-2]  
    }    return dp[n]  
}
```
