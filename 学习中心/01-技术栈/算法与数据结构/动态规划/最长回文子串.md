# 解题思路
## 思考子问题
- 如果字符串长度为1的话，一定为回文串
- 如果字符串长度为2的话，则必须两个字符相等，为回文串
- 如果字符串长度为3的话，第一个和最后一个字符相等，为回文串
- 如果字符串长度为4的话，对应的两个字符必须相等，为回文串

一个回文串，去掉两端的字符，仍然为回文串。

当`s[i] == s[j]`时，若`s[i+1:j-1]`为回文串或`j-i+1 < 4`则`s[i:j]`为回文串。

# 递归公式

$$
dfs(i,j) = (s[i] == s[j]) \&\& (dfs(i+1, j-1) || (j-i+1) < 4)
$$
# AC代码
```go
func longestPalindrome(s string) string {  
    m := len(s)  
    if m < 2 {  
       return s  
    }  
  
    f := make([][]bool, m+1)  
    for i := 0; i <= m; i++ {  
       f[i] = make([]bool, m+1)  
       f[i][i] = true  
    }  
  
    max := 1  
    begin := 0  
  
    for j := 0; j < m; j++ {  
       for i := 0; i < j; i++ {  
          if s[i] == s[j] && (f[i+1][j-1] || (j-i+1) < 4) {  
             f[i][j] = true  
          }  
  
          if f[i][j] && j-i+1 > max {  
             begin = i  
             max = j - i + 1  
          }  
       }    }    return s[begin : begin+max]  
}
```
