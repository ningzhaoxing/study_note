题目：
给定两个字符串，`s` 和 `p`，求字符串 `s` 中所有 `p` 的异位词子串，并记录其起始索引。

思路：
1. 用 `cnt` 数组记录字符串 `p` 每个字母的出现次数。
2. 对 `s` 做滑动窗口，逐渐右移窗口右边界。
3. 每当一个字母进入窗口， 则在 `cnt` 数组中的对应字母出现次数 -1。
4. 当 `cnt` 数组中该字母出现次数<0时，循环左移窗口左边界，*并恢复从窗口出去的字母出现次数*，直到该字母出现次数=0
5. 循环结束后，如果此时的窗口大小=`p`的长度，则是一个异位词子串

代码：
```go
func findAnagrams(s string, p string) []int {
    var cnt [26]int
    for _, c := range p {
        cnt[c-'a']++
    }
    ans := make([]int,0)
    l:=0
    for r:=0;r<len(s);r++{
        c := s[r]-'a'
        cnt[c]--
        for cnt[c]<0 {
            cnt[s[l]-'a']++
            l++
        }
        if r-l+1==len(p){
            ans=append(ans,l)
        }
    }
    return ans
}
```
