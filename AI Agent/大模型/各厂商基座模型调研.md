以下七家大模型厂商均支持：
字节、阿里、DeepSeek、Kimi、智谱、Minimax、百度
只需提供 `base_url`（指向厂商的 API 地址）和 `api_key` 进行配置。

> 备注：
> 大模型必须具备工具调用能力，视觉理解能力可选（系统建模 Agent 需要用到视觉理解能力）

以下是实际测试效果好的模型（**测试过、支持视觉理解、工具调用**）：

| 厂商   | 推荐模型         | 备注                              |
| ---- | ------------ | ------------------------------- |
| 阿里千问 | qwen3.5-plus | 1. 支持视觉理解、工具调用<br>2. 上下文长度 1M   |
| 月之暗面 | KIMI-K2.5    | 1. 支持视觉理解、工具调用<br>2. 上下文长度 256K |
